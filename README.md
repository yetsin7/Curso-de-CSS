# Curso de CSS — De cero a interfaces limpias, responsivas y mantenibles

Repositorio oficial: <https://github.com/yetsin7/Curso-de-CSS>

## Documentacion de referencia

Consulta [Documentación-CSS.md](./Documentación-CSS.md) para estudiar CSS con enlaces oficiales y
una ruta de temas recomendada.

## Para que sirve este curso

Este curso crece mas alla del CSS basico. La meta es que aprendas tanto el
fundamento como el uso real para landing pages, catalogos, dashboards y piezas
comerciales modernas.

## Modulos del curso

1. `01-introduccion-y-sintaxis`
2. `02-box-model-y-espaciado`
3. `03-tipografia-colores-y-fondos`
4. `04-flexbox`
5. `05-grid-y-responsive`
6. `06-proyecto-final-css`
7. `07-position-y-z-index`
8. `08-formularios-y-tablas`
9. `09-pseudoclases-y-estados`
10. `10-transiciones-y-animaciones`
11. `11-arquitectura-css-y-reutilizacion`
12. `12-css-para-ecommerce-y-dashboard`
13. `13-optimizacion-responsive-real`
14. `14-proyecto-final-css-pro`

## Resultado esperado

Al terminar deberias poder vestir una interfaz completa para web moderna y
hacer que funcione bien en movil y desktop.
