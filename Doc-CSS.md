# Documentacion de CSS

CSS significa Cascading Style Sheets, u hojas de estilo en cascada. Sirve para
darle apariencia visual a una pagina web: colores, tamanos, espaciado, layout,
animaciones y comportamiento responsive.

## Documentacion recomendada

- [MDN CSS Docs](https://developer.mozilla.org/en-US/docs/Web/CSS)
  Es la referencia mas practica para aprender propiedades, selectores y ejemplos.
- [W3C CSS](https://www.w3.org/Style/CSS/)
  Es una fuente formal sobre los estandares relacionados con CSS.

## Temas principales que debes estudiar

1. Selectores CSS:
   permiten elegir que elementos vas a estilizar.
2. Colores y tipografia:
   definen la identidad visual y la legibilidad.
3. Box model:
   explica como ocupa espacio un elemento en pantalla.
4. Margin, padding y border:
   controlan separacion interna, externa y bordes.
5. Display y position:
   determinan como se comporta un elemento dentro del layout.
6. Flexbox:
   sistema para alinear y distribuir elementos en una fila o columna.
7. Grid:
   sistema para construir layouts bidimensionales mas complejos.
8. Responsive design:
   adaptar la interfaz a movil, tablet y desktop.
9. Media queries:
   reglas que cambian estilos segun el tamano de pantalla u otras condiciones.
10. Animaciones y transiciones:
   agregan movimiento y cambios visuales suaves.
11. Variables CSS:
   permiten reutilizar colores, medidas y tokens visuales.
12. Pseudo-clases y pseudo-elementos:
   ayudan a estilizar estados como `hover`, `focus` o partes especificas de un
   elemento.

## Orden recomendado

1. [Fundamentos de CSS](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Styling_basics)
   Aqui entiendes la sintaxis y la idea general del lenguaje.
2. [Selectores CSS](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_selectors)
   Aprendes a elegir correctamente que estilizar.
3. [Box model](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Styling_basics/Box_model)
   Este concepto te evita muchisimos errores visuales.
4. [Flexbox](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/CSS_layout/Flexbox)
   Muy util para menus, cards y distribucion simple.
5. [Grid](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/CSS_layout/Grids)
   Ideal para layouts mas completos.
6. [Responsive design](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/CSS_layout/Responsive_Design)
   Te ensena a hacer interfaces utiles en diferentes pantallas.

## Ejemplo basico

```css
body {
  font-family: Arial, sans-serif;
  background-color: #f4f4f4;
  color: #222;
}

h1 {
  color: #0066cc;
}

p {
  font-size: 18px;
}
```
